Overview

Solid Buttons

"Solid" buttons are the default button-type.

Change the color by adding one of the classes: .primary, .secondary, .success, .warning, .danger


Outline Buttons

Add the additional .outline class to create Outline-Type Buttons


Rounded Solid Buttons

Appending either the.round or .rounded class creates fully rounded button-types


Rounded Outline Buttons

All of the above classes can be mixed to create color and style variations.




Quick note on class heirarchy

From least dominate to most dominate, class(style) heirarchy is as follows:

-> button, .button, [type=button]                         ...
   -> .primary, .secondary, .success, .warning, .danger   ...
     -> .outline                                          ...
        -> .round, .rounded (overrides all other classes)
