Installation Step 1

NPM


  npm install scss-button-mixins --save

  npm install

Yarn


  yarn add scss-button-mixins

Installation Step 2

Inside your stylesheet:

Somewhere like: /src/styles/main.scss ...


  @import '../path/to/node_modules/scss-button-mixins/stylesheets/scss-button-mixins';
All Done.

You're ready to start using it.
